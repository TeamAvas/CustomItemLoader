# CustomItemLoader
[Plugin] PocketMine-MP adds items using ItemComponent Packet.
<br>

# Description
You can implement the items and tools you want!
<br>
To implement this, I referred to [CustomItem](https://github.com/alvin0319/CustomItem)!
